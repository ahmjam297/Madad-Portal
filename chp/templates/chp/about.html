CEMK-Help Portal A Python Django Project

Project Description

CEMK STUDENT HELP PORTAL is a web portal on which various types of queries and complaints related to various field such as Academic. Hostel, Civil, Library, Sports, and Ragging etc. can be made by college students. In the current system all the activities are done manually. It is very time consuming and costly. The CEMK Student Help Portal will be helpful for students and staffs as well. It will make the process more accountable,Smooth and efficient.

Project Details

  • Django Web-Framework
  • SQLite3
  • HTML,CSS,Bootstrap3
